# Ninjafy-cli

Converts a string into its Ninja-style equivalent on the command-line

# Install

```
$ npm install --global ninjafy-cli
```

## Usage

```
ninjafy --help

  Converts a string into its Ninja-style equivalent on the command-line

  Usage
    $ ninjafy <input>

  Examples
    $ ninjafy John Doe
    Zumorito Temoku
```
